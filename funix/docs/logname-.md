NAME
====

logname-(1f) - \[FUNIX\] display login name (LICENSE:PD)

SYNOPSIS
========

logname- \[ **-help**\|**-version**\]

DESCRIPTION
===========

Demonstrate call to system\_getlogin(3f) (which calls getlogin(3c)

OPTIONS
=======

****--help****

:   display command help and exit

****--version****

:   output version information and exit

EXAMPLE
=======

Command usage:

            logname-
