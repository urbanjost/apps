NAME
====

rm-(1f) - \[FUNIX:FILESYSTEM\] remove file(s) (LICENSE:PD)

SYNOPSIS
========

rm- file(s) \| \[-**-help**\|**--version**\]

DESCRIPTION
===========

Remove file(s) by calling remove(C).

OPTIONS
=======

**file(s)**

:   list of files to remove

**help**

:   display help and stop

**version**

:   display version and stop

EXAMPLE
=======

rm- \*.o
