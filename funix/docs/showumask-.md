NAME
====

showumask-(1f) - \[FUNIX\] show umask in decimal, octal, hex, and binary
(LICENSE:PD)

SYNOPSIS
========

showumask- \[ **-help**\|**-version**\]

DESCRIPTION
===========

example program calling system\_getumask(3f) to get umask value.

EXAMPLE
=======

showumask-

        18 O'0022' Z'12' B'000000010010'
